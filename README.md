# Will's Budget App

GitHub Pages-ready budget app.

## Upload to GitHub
1. Create a new public repository.
2. Upload these files to the root of the repo: index.html, manifest.json, sw.js, icon.svg, README.md.
3. Go to Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Open the Pages URL on iPhone in Safari.
7. Share > Add to Home Screen.
